# RutinKu

Aplikasi habit tracker sederhana untuk membantu pengguna membangun rutinitas harian.

## Fitur Utama

- Tambah & kelola habit harian
- Tracking progress mingguan

2. Buka file APK di HP Android
3. Izinkan instalasi dari sumber tidak dikenal jika diminta
4. Buka aplikasi RutinKu
